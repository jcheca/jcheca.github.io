Selector CSS3

[attr|=value]: Selecciona todos los elementos que tengan un atributo attr cuyo valor comience por la cadena completa "value". Después de esta cadena no puede haber otro carácter, salvo un espacio en blanco o un guión.

[id|=parrafo]: Selecciona todos aquellos elementos cuyo atributo id comienza por la palabra completa "parrafo":

Párrafo 1 (id="parrafo").

Párrafo 2 (id="segundo-parrafo").

Párrafo 3 (id="parrafo-tercero").

Párrafo 4 (id="cuartoParrafo").

Párrafo 5 (id="quintoparrafo").

Párrafo 6 (id="parrafoUltimo").
