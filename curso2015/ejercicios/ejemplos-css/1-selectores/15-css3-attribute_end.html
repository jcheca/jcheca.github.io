Selector CSS3

[attr$=value]: Selecciona todos los elementos que tengan un atributo attr cuyo valor termine con la cadena "value".

[id$=parrafo]: Selecciona todos aquellos elementos cuyo texto en el atributo id termina con el texto "parrafo":
